# homebrew-doppel

Homebrew tap for [Doppel](https://github.com/ishm6m/doppel) — a 100% offline macOS
duplicate & near-duplicate file finder.

```sh
brew install --cask --no-quarantine ishm6m/doppel/doppel
```

`--no-quarantine` is needed because Doppel is open source and ad-hoc signed (not
notarized — that requires a paid Apple Developer account). Update with `brew upgrade`.
